## System
### Status
Get system status related information.


```shell
curl "https://api.peachtopeach.com/v1/system/status"
```

> The above command returns JSON structured like this:

```json
{
  "status": "online",
  "date": "2021-09-17T14:41:41.605Z"
}
```
#### HTTP Request
`GET /v1/system/status`
