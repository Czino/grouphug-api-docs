# System
## Status
Get system status related information.

```shell
curl https://grouphug.peachbitcoin.com/v1/system/status
```

> The above command returns JSON structured like this:

```json
{
  "error": null,
  "status": "online",
  "serverTime": 1692788403879
}
```

### HTTP Request
`GET /v1/system/status`
